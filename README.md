# test-server 
A simple server for the BGL pre-interview task, bulit with Go

## Requirements
- Go > 1.7 
- [go-chi](https://github.com/go-chi/chi)


## How to run this
**1. Clone This Repo**

    go get https://github.com/Newo6174/test-server
    cd test-server
    
**2. Run the server on port 8080**

    go run server.go
    
**3. Check if the server is running**

    curl 127.0.0.1:8080
